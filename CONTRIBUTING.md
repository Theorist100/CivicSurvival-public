# Contributing to Civic Survival

Thank you for your interest in the mod. Please read this before opening anything.

## Reporting bugs and suggesting features

**All bug reports and feedback go to our Discord, not GitHub Issues:**

➡️ <https://discord.gg/yg4G2rVrd> (also reachable from the in-game Roadmap panel)

When reporting a bug, please include:
- the game version and the mod version,
- steps to reproduce,
- what you expected vs. what happened.

This repository is published for transparency and code auditing. It is a **read-only
mirror of release snapshots**, not the development repository — development happens
privately, and the public repo is updated one snapshot per release.

based on the software, so submitting a derivative as a PR is outside the rights the
license grants. The author is also the sole copyright holder and dual-licenses the
same code to Paradox separately, which requires retaining full ownership of all
contributed code.

If you have a fix or feature you would like considered, **describe it on Discord**. If
the author wants to incorporate a code contribution, it can only be accepted under a
separate, explicit written agreement (a contributor license assignment) that transfers
the necessary rights — otherwise it cannot be merged. Please do not open a PR
expecting it to be merged as-is.

## Localization

Translations are very welcome — the best way to contribute. The mod ships:
- `CivicSurvival/Localization/en-US.json` — English (neutral)
- `CivicSurvival/Localization/uk-UA.json` — Ukrainian (realistic crisis context)

If you would like to add or improve a locale, reach out on
[Discord](https://discord.gg/yg4G2rVrd) — translation contributions are handled there
and folded into a future release snapshot.
